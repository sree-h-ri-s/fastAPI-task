# fastAPI-task

- Create virtual environment and install the requirements.txt

  ```
  pip install -r requirements.txt
  ```
  
- Run the server

    ```bash
    python server.py
    ```

- Run the client

    ```bash
    python client.py
    ```
- Send message using the input and can see the output on the client side in json format
